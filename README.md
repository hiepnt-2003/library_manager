# Library Manager Spring Boot Project

This project is a basic Spring Boot application for managing a library, using MySQL as the database.

## Features
- Spring Boot 3.2.x
- MySQL integration
- JPA for data access

## Getting Started
1. Update `src/main/resources/application.properties` with your MySQL credentials.
2. Run MySQL and create a database named `library_db`.
3. Build and run the project:
   - `mvn spring-boot:run`

## Project Structure
- `src/main/java/com/example/library_manager` - Main source code
- `src/main/resources` - Configuration files
- `src/test/java/com/example/library_manager` - Test code
